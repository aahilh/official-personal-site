Inventory Management System

The system I made is a full stack application that utilized React JS, Spring Boot, and MySQL. This application provides a listing of all products a distributor or store might have to offer. It also provides special access for users that want to manipulate products in display. It showcases the use of CRUD operations, which are important when working with backend.


Pokemon Weather App

This GUI application was made using JavaFX and its role is to read from two REST APIs to provide the user a response in accordance to the real time weather in that region. It determines if the weather would be suitable for a Pokemon that would inhabit the selected region. This project demonstrates the use of threads and processing data from APIs to form an output for the user.
